Authentication Products

BlokKey — Mobile Passwordless MFA

Knowledge Workers / Office Staff

Push notification to registered device → Biometric confirmation → Intent verification → Authenticated. No password at any point.

Key tech: HD wallet generates unique ECDSA key pair per application. Private keys never leave device. Per-app isolation prevents cross-app compromise.

Biometric Device Intent = 3-Factor MFA

BlokBadge — Deviceless MFA

Frontline / Floor Workers

Scan printed QR badge at terminal → Enter personal PIN → Authenticated. No smartphone, no hardware token, works with gloves.

Key tech: Split-key cryptography. Badge holds Key Fragment A, server holds Key Fragment B. Neither usable alone. Badge prints in 30 seconds, deactivates instantly.

Possession (Badge) Knowledge (PIN) = 2-Factor MFA

Cryptographic Foundations

ECDSA Asymmetric Crypto

  • No shared secrets between client and server
  • Private keys never transmitted over network
  • Eliminates MITM and replay attacks
  • Server compromise yields nothing usable

Split-Key Security (BlokBadge)

  • Badge: encrypted partial key (Fragment A)
  • Server: complementary key (Fragment B)
  • Both fragments required for auth
  • Lost badge = useless without server half

HD Wallet Isolation (BlokKey)

  • Unique key pair per application
  • Hierarchical Deterministic derivation
  • Compromise of one app ≠ compromise of others
  • Keys derived, not stored in bulk
