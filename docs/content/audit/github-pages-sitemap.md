# GitHub Pages Resource Hub Sitemap

## Goal
Create a focused resource hub that helps BlokSec win with SMB and mid-market buyers that have frontline workers, shared devices, compliance pressure, and recurring password-reset friction.

## Primary navigation
- Home
- Industries
- Partners
- Research
- Tools
- Audit

## Home
Purpose:
- explain the wedge in plain English
- route visitors by role or industry
- surface the strongest research, tools, and next-step assets

Recommended sections:
- hero: frontline/shared-device passwordless GTM hub
- updated focus: SMB + mid-market + frontline-heavy teams
- why now: compliance, phishing, reset burden, turnover
- choose your path: industry, partner, research, audit
- featured tools: ROI calculator, partner one-pager, technical overview

## Industries
Priority pages:
- Manufacturing
- Retail
- Warehousing and Logistics
- Food Processing
- Care Settings

Each page should answer:
- what makes authentication hard here
- why passwords and phone-based MFA fail
- what compliance or audit pressure exists
- what BlokSec angle matters most
- which proof points or assets to use next

## Partners
Priority pages:
- MSP channel strategy
- partner program outline
- partner enablement kit
- competitive responses

Primary outcome:
help an MSP quickly understand fit, margin, pain, and what to sell.

## Research
Priority pages:
- research revisit packet
- strategy framework
- frontline passwordless market research
- AI tooling x GTM funnel matrix
- Benefits, ROI & switching narrative (stakeholder discussion doc)
- market, compliance, product, and competitor context

Primary outcome:
make the thesis easy to defend internally and externally.

## Tools
Priority assets:
- ROI calculator
- technical overview
- technical webslides
- MSP one-pager
- internal strategy one-pager
- source deck download

## Audit
Priority pages:
- research revisit
- GTM vs product capability matrix
- site audit summary
- GitHub Pages sitemap
- next actions

## Audience paths
### Path 1: founder or GTM lead
Home -> Audit -> Sitemap -> Next Actions

### Path 2: partner or MSP
Home -> Partners -> MSP strategy -> ROI calculator -> one-pager

### Path 3: industry-specific campaign build
Home -> Industries -> selected vertical -> compliance context -> tool or case-study follow-up

### Path 4: website rewrite or content build
Home -> Research -> positioning -> market -> competitors -> playbooks
