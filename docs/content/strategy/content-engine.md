# Content & SEO Engine

## Strategy

Own the "frontline worker authentication" content space. No competitor has claimed it.

## Three Repeatable Content Types

### 1. Breach Analysis Series (Weekly)
- Newsjack data breaches
- Show how passwordless would have prevented each
- Target: "[Company] data breach" + "credential theft prevention"
- Time: ~2 hours/week

### 2. Compliance Deep-Dive Guides (Monthly)
- Framework-specific guides with checklists
- Target: "[Regulation] MFA requirements [year]"
- Lead magnets for email capture
- Time: ~4 hours/month

### 3. Industry Landing Pages (Quarterly)
- Vertical-specific conversion pages
- Target: "[industry] authentication solutions"
- Refresh quarterly with new data
- Time: ~3 hours/quarter

## Content Pillars for SEO

1. **Frontline Worker Authentication Hub** — pillar page + 10-15 supporting posts
2. **Compliance Resource Center** — framework-specific guides
3. **Security Intelligence** — breach analysis, threat trends
4. **MSP Partner Resources** — public-facing partner content

## DocSmith Opportunity (Selective)

Build docs for 2-3 auth-relevant repos only:
- **Keycloak** (~25K stars) — Direct auth relevance
- **Authentik** (~15K stars) — Identity provider space
- **ory/kratos** (~12K stars) — Identity management

Include "Securing [Tool] with Passwordless Authentication" sections. Do NOT build docs for general security/pen testing tools.
