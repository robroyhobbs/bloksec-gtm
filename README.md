# BlokSec GTM

This repository is the working GTM home for BlokSec.

It combines three things in one place:
- source-of-truth strategy and market context
- reusable sales, partner, and content assets
- a GitHub Pages-ready resource hub published from `/docs`

## Repo structure

- `PRODUCT.md` — stable product and GTM brief
- `docs/` — GitHub Pages source for the resource hub
- `docs/content/context/` — company, product, market, compliance, pricing, and competitor context
- `docs/content/strategy/` — GTM plan, target markets, content engine, and MSP strategy
- `docs/content/research/` — deeper research documents migrated from prior work
- `docs/content/playbooks/` — repeatable outreach and content templates
- `docs/content/industry/` — industry-specific angle pages for frontline-heavy targets
- `docs/content/audit/` — current audit packet, sitemap, and next actions
- `docs/tools/` — ROI calculator and HTML sales/technical collateral
- `docs/assets/` — source decks and future downloadable files

## Publishing model

The resource hub is intended to publish through GitHub Pages using the `/docs` folder as the site source.
